Um projeto simples em JavaScript que classifica o ranking de um herói com base na quantidade de XP. 
Este desafio foi criado para praticar:
- Estruturas de decisão
-  - Arrays e objetos
   -  - Funções -
      -  Validação de dados
      -  - Recursos modernos do JavaScript
         -  O sistema determina automaticamente o nível do herói de acordo com seus pontos de experiência.
